Setup and running:
0) Install git
1) Install Mongo
2) Install python
3) Install virtualenv
4) Install python-pip
5) Create a new virtualenv by typing "virtualenv env" on terminal/command prompt
6) Source into the virtualenv (Mac: "source env/bin/activate" or Windows "venvname\Scripts\activate") on terminal/command prompt
7) Clone our repository (your current location in terminal/command prompt should be beside the venv folder)
8) Type "git clone https://github.com/bhargavrpatel/Travelabulous---nonworking.git"
9) CD into the project where requirements.txt, test.py and runserver.py are located
10) Install the project requirements by typing "pip install -r requirements.txt"
11) Type "python fixtures.py" to add sample questions and guest user
12) Type "pyton test.py" to test the database subsystem. Refer to the code to see the assertions
13) Type "python runserver.py" to start the web server
14) Navigate to localhost:5000/questionnaire
